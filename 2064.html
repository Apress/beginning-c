6

 

The current C standard require the return type for the main() function to be type int. Therefore the code for the program on Page 6 should be:

 

#include <stdio.h>

int main()

{

  printf("Hello world");

  return 0;

}

Of course, this also applies to all subsequent examples in the book where the return type for main() has been specified as void. In each case the return type for main() should be changed to int and a return statement as in the example above inserted in the body of main().
